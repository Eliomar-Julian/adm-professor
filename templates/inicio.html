Bem vindo ao programa de gestão escolar

Este é um programa escolar de gestão de alunos voltado ao professor para que o mesmo consiga gerenciar as notas e atividades de seus alunos de maneira rápida e prática sem usar um monte de papéis desnecessários.
ADICIONE, ALTERE, IMPRIMA, ORGANIZE, sem preocupação ou dor de cabeça.
Para acessar as diferentes áreas do programa navegue pelo menu lateral à esquerda.
Deixe que o computador trabelhe para você nessa árdua tarefa de passar conhecimentos.
Se mantenha atualizado, faça sempre um git pull para manter seu repositório atualizado.
